About This Game

What Is This?

This is a fun and interactive Rock-Paper-Scissors game built using HTML, CSS, and JavaScript. It's a browser-based version of the classic game, with score tracking and instant results.

Who Made It?

Hi, I'm Vishal Yadav — a developer passionate about creative web interfaces and interactive experiences. This was built as a fun learning project!

How It Works

Select Rock, Paper, or Scissors — the computer makes a random choice and the winner is shown instantly. First to 5 points wins the match!

Tech Stack

  • HTML5 & CSS3
  • Vanilla JavaScript
  • Responsive Layout
  • Custom SVG background

Future Features

  • Multiplayer support
  • Sound effects
  • Animated hand gestures
